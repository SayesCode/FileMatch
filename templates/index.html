Welcome to the FileMatch API! Verify if a file's MIME type matches its extension.

Access Here to Test

About

The FileMatch API provides a robust way to validate file types without saving them. It's perfect for ensuring data integrity in your applications.

How to Use

To use the API, send a POST request to the following endpoint:

POST /api/v1/verify/validate-file/

Headers:

  • Content-Type: multipart/form-data

Body:

  • file: The file you want to validate (required).

Example Request


curl -X POST "http://127.0.0.1:8000/api/v1/verify/validate-file/" \
-F "file=@example.jpg"

Example Response


{
  "status": "success",
  "message": "The file 'example.jpg' is valid."
}
